## 🎨 Visual Design & Aesthetics
This website features a glassmorphic tech-focused design layout tailored for modern developers:
* **Glassmorphism Elements:** Translucent layout containers, blur filters, and neon-glowing accents.
* **Light / Dark Mode Toggle:** Selection is saved and persists on refresh using `localStorage`.
* **Micro-Animations:** Fluid layout shifts, floating badge actions (React, Java), typing typewriter animations, and custom scroll bars.

---

## ✨ Interactive Features

1. **Custom Loading Screen:** Initial fade-in reveal with loading bar progress.
2. **Hero Role Typewriter:** Auto-typing role indicators ("Fullstack Developer", "Java Coder (DSA)", "MERN Stack Intern", "Problem Solver").
3. **Interactive Tabs (About Section):** Dynamic toggles for Story & Goals, Experience timeline, and Interests.
4. **On-Scroll Skills Fills:** Uses the **Intersection Observer API** to animate skills percentage indicators and progress bars when scrolled into view.
5. **Project Filter Control:** Dynamically filter cards by categories (*Fullstack / MERN*, *Java / DSA*) with scale-fade transitions.
6. **Form Validation & Modals:** Complete frontend contact validation triggering a custom victory modal notification rather than alert popups.
7. **Scroll-to-Top Button:** Floats in after scrolling 400px down the page.
